# Titanic Survival Predictor (84% Accuracy)

A Machine Learning project using Python to predict passenger survival outcomes from the Titanic shipwreck.

## Key Insights Found:
- **Gender Gap:** Female passengers had a significantly higher survival rate.
- ![Titanic Survival Analysis](titanic_graph.png)
- **Class Factor:** Survival was not just luck; passenger class (Pclass) and ticket cost were major predictors.
- **Model Performance:** Achieved 84% accuracy using feature engineering on Sex and Pclass.

## Tech Stack:
Python, Pandas, Scikit-Learn, Matplotlib/Seaborn.
